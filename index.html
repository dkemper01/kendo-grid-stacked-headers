Kendo Labs : Grid :: Stacked Headers

  • Notes

  • This is not a one-off implementation of multi-column headers
  • Each and every header is sortable, filterable, and groupable
  • All other grid operations function normally
  • Due to internal grid methods such as leafDataCells, and its usage within _createResizeHandle, the resize handle is only drawn for the header cells on the bottom row.
